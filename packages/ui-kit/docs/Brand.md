## 2. Typography

### Typeface Hierarchy

| Role | Font Family | Fallback | Style | Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Headers** | **PP Gosha Bold** | Space Grotesk, Sans-serif | Bold, Sharp angles | Logo, Page Titles, specific Brand headers. |
| **Subheaders** | **PP Gosha** | Space Grotesk, Sans-serif | Regular | Subtitles, Quotes, Emphasis. |
| **Body** | **Roboto Regular** | Inter, Sans-serif | Regular | All body text, UI elements, descriptions. |

> **Note on Implementation:** If `PP Gosha` (paid license) is unavailable for web use, `Space Grotesk` is the approved open-source alternative for headers. `Inter` is often used as a modern alternative to `Roboto` in our web apps, but `Roboto` is the official brand standard.

### Usage Rules
*   **Headers:** Use `PP Gosha Bold` for high-impact text.
*   **Legibility:** Use `Roboto` (or `Inter`) for long-form content to ensure maximum readability.
*   **Pairing:** Sharp headers (`Gosha`) contrast well with neutral body text (`Roboto`).
