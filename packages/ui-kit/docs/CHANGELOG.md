# Changelog

## 0.1.0 (2025-12-19)

### Added
- Initial project structure with Vite, TypeScript, and Tailwind CSS.
- Design Tokens (Brand colors, Typography) in `tailwind.config.js`.
- **Atoms**: `Badge`, `Button`, `Input`, `Logo`, `ProgressBar`, `Select`.
- **Molecules**: `AnnouncementBanner`, `AuthDivider`, `ReportIssueCard`, `SidebarItem`, `SocialLoginButton`, `StatCard`, `TaskListItem`.
- **Organisms**: `AuthCard`, `DashboardHeader`, `ImpactWidgets`, `LoginForm`, `Sidebar`, `TaskList`.
- **Templates**: `DashboardLayout`.
- **Pages**: `AuthPage`, `VolunteerDashboardPage`.
- Comprehensive Storybook stories for all components.
- Unit tests with Vitest and React Testing Library.
